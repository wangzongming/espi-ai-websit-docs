  - title: 完整可靠
    details: ESP-AI 作为面向生产环境发布的框架，有完整的 IAT(ASR) ➡️ LLM/RAG ➡️ TTS，每一个步骤都可以定制。

  - title: 配置化和零代码
    details: 除特殊业务编写的插件外，框架或插件提供的配置一般能满足大部分场景。后续会提供配置生成服务，将会完全零代码。
