ESP-AI 通信协议是一套专为 边缘 AI 设备 与 云端服务 设计的高性能通信标准。基于统一的协议规范，无论客户端 / 服务端采用何种编程语言实现，均可实现与任何兼容 ESP-AI 标准的 智能硬件 或 云服务 的无缝通信。

- 数据传输方向：全双工
