`v4.0.0` 开发板和 `v3.0.0` 相比： 

- 更换数字位模拟麦，提升稳定性和生产难度。
- 更换max98375为ES8311，提升稳定性和生产难度。
- 新增音量调节拨盘。
- 新增3.5mm耳机接口。
- 调整 TFT 屏幕接口IO顺序，以适配更多市面上面的屏幕，官方默认使用2.4寸屏幕。
