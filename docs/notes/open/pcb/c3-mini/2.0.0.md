## 功能 
- ✨ 采用 ESP32-C3 芯片 + ES8311 音频编解码
- ✨ 支持 ESP-AI 开放平台所有服务
- ✨ 板载语音唤醒芯片，支持在线更改唤醒词
- ✨ 支持触摸唤醒、按钮唤醒
- ✨ 支持长按开关机、一键开关机
- ✨ 支持 OLED 屏幕
- ✨ 外接所有其他元件，方便放在玩偶中
