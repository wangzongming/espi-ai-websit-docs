ビジネス データは、サーバー ラテラル ストレージとクライアントラテラル ストレージの 2 方法。つまり、Nodejs または Arduino を使って、ハードウェア内のビジネスしたりできます。

最適な方法を選択する方法。

## Arduinoコード操作

詳細については、[getLocalData / setLocalData](/config-client/instance/#onbinddevice)をご参照ください。

```c
// 设置数据
setLocalData("ext1", "自定义数据xxx");

// 读取数据
String ext1 = getLocalData("ext1");

```

## Nodejs コード操作
現在、サーバーはSETのみをサポートしており、読み取ったりはサポートしておりません。ただし、サービスは多くのデバイスをマウントするため、ハードウェアIDは複数あるだけです。

詳細については、[setLocalData](/config-server/instance/#setlocaldata)をご参照ください。

```c
const espAiIns = espAi({ ... });
await espAiIns.setLocalData("xxx-xxx-xxx-xxx", "ext1", "业务数据");
```
