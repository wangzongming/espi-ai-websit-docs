## 照明の説明

直接使用される `ESP-AI` グループみ込み方向の色については、[指示灯](/example/led/) を参照してください。ただし、内赵の表示灯に记づいて以下の表示灯が追加されます。


|ライトステータス |色 |説明 |タイプ |
| -------- | ------ | -------------- | -------- |
|指摘する | べっ甲イエロー |写真べっ甲イエロービジネスサービスが开かれ続かない|システムエラー|


## グループみ込みの音声コマンド

のコマンドは設定の必要がなく、すでに组み込まれています。

|コマンド |返信 |注意事項 |
| ----------------- | ---------------------------- | ------------------------------ |
|音量最大 xx% | OK |声の大きさ:音量最大 60% |
|より声で/より大で | OK |音量コントロール |
|声を下げて/声を下げて | OK |音声コントロール音量 |
戻る。|冬眠|
